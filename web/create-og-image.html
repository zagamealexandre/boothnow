How to create your social media image:

  1. Right-click on the blue image above
  2. Select "Save image as..." or "Copy image"
  3. Save it as og-image.png in your web/public/images/ folder
  4. Make sure the image is exactly 1200x630 pixels
  5. Test the link sharing on Messenger, WhatsApp, etc.

Alternative: Use a design tool

You can also create a more professional image using:

  • Canva: Use their social media post template (1200x630)
  • Figma: Create a custom design with your branding
  • Photoshop: Design a high-quality image

Image requirements:

  • Size: 1200x630 pixels (1.91:1 ratio)
  • Format: PNG or JPG
  • File size: Under 1MB for best performance
  • Include your KUBO logo and branding
  • Make it readable on mobile devices
